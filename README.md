# datasci-wind-api
This code is use for query NOAA dataset which includes weather data.

## How to run

### 1.Request for token
This will send token to your email.
https://www.ncdc.noaa.gov/cdo-web/token

### 2.Running
copy your `apiKey` and paste them in the code. Running with
`yarn start`
